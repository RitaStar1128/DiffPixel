Privacy Policy

Effective date: May 27, 2025  ·  DiffPixel Chrome Extension

DiffPixel does not collect, transmit, or share any personal data. All data created by the extension stays entirely on your device.

1. Overview

DiffPixel ("the Extension") is a pixel-perfect overlay comparison tool for designers and developers. It allows you to display reference images as a transparent overlay on browser tabs so you can visually compare designs against live web pages.

We are committed to protecting your privacy. This policy explains what data the Extension handles and why.

2. Data We Do Not Collect

The Extension does not collect, store, transmit, or share:

  • Personal information (name, email, IP address, etc.)
  • Browsing history or visited URLs beyond what is needed in the moment to scope your settings
  • Keystrokes, clicks, or mouse activity
  • The content of web pages you visit
  • The images you load as overlays

3. Local Storage

The Extension uses chrome.storage.local exclusively to save your preferences on your own device:

  • UI preferences – theme (dark / light) and display language (EN / JA)
  • Per-domain overlay settings – opacity, position, scale, blend mode, and layer configuration for each site you use the Extension on
  • Image data – reference images you load are stored locally as Base64 data to persist across sessions

All of this data exists only in your browser's local storage. It is never sent to any server.

4. Permissions Explained

  • activeTab – to identify the tab you are currently viewing so the overlay can be injected into it
  • scripting – to inject the overlay UI into the active tab on demand
  • storage – to read and write your settings locally
  • unlimitedStorage – to accommodate large or multiple reference images without hitting the default storage quota

No permission is used to access, read, or transmit information about your browsing activity or the content of pages you visit.

5. Third Parties

The Extension does not integrate with any third-party analytics, advertising, or data services. No data is sold, transferred, or shared with any third party.

6. Remote Code

The Extension does not load or execute any remote code. All scripts and styles are bundled within the extension package and governed by a strict Content Security Policy (script-src 'self').

7. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect data from children.

8. Changes to This Policy

If this policy is updated, the new version will be published at this URL with a revised effective date. Continued use of the Extension after changes constitutes acceptance of the updated policy.

9. Contact

Questions or concerns? Please open an issue on the GitHub repository.

プライバシーポリシー

施行日: 2025年5月27日  ·  DiffPixel Chrome 拡張機能

DiffPixel は個人データを収集・送信・共有しません。拡張機能が作成するすべてのデータはお使いのデバイス上にのみ保存されます。

1. 概要

DiffPixel（以下「本拡張機能」）は、デザイナーおよびデベロッパー向けのピクセルパーフェクトなオーバーレイ比較ツールです。参照画像をブラウザタブ上に半透明のオーバーレイとして表示し、デザインと実際のウェブページを視覚的に比較するために使用します。

本ポリシーでは、本拡張機能が扱うデータと、その目的について説明します。

2. 収集しないデータ

本拡張機能は以下のデータを収集・保存・送信・共有しません：

  • 個人情報（氏名・メールアドレス・IPアドレスなど）
  • 閲覧履歴やアクセスした URL（設定のスコープ限定に必要な瞬時の参照を除く）
  • キーストローク・クリック・マウス操作
  • 閲覧したウェブページのコンテンツ
  • オーバーレイとして読み込まれた画像の内容

3. ローカルストレージ

本拡張機能は chrome.storage.local のみを使用し、以下の設定をお使いのデバイス上に保存します：

  • UI設定 – テーマ（ダーク／ライト）および表示言語（EN／JA）
  • ドメインごとのオーバーレイ設定 – 不透明度・位置・スケール・合成モード・レイヤー構成
  • 画像データ – 読み込んだ参照画像は Base64 形式でローカルに保存されセッションをまたいで維持されます

これらのデータはすべてブラウザのローカルストレージにのみ存在し、いかなるサーバーにも送信されません。

4. 権限の説明

  • activeTab – オーバーレイを注入するために現在表示中のタブを特定するため
  • scripting – アクティブなタブにオーバーレイ UI を注入するため
  • storage – 設定をローカルに読み書きするため
  • unlimitedStorage – 大容量または複数の参照画像を保存するためにデフォルトのストレージ上限を超える容量を確保するため

いずれの権限も、閲覧アクティビティやページコンテンツへのアクセス・取得・送信には使用しません。

5. 第三者への提供

本拡張機能はサードパーティの分析・広告・データサービスと連携しません。データを第三者へ販売・転送・共有することはありません。

6. リモートコード

本拡張機能はリモートコードを読み込み・実行しません。すべてのスクリプトとスタイルは拡張機能パッケージ内にバンドルされており、厳格なコンテンツセキュリティポリシー（script-src 'self'）によって管理されています。

7. 子どものプライバシー

本拡張機能は13歳未満の子どもを対象としていません。子どものデータを意図的に収集することはありません。

8. ポリシーの変更

本ポリシーを改訂した場合、改訂後の版を同 URL で公開し、施行日を更新します。変更後も引き続き本拡張機能を使用することで、改訂されたポリシーに同意したものとみなします。

9. お問い合わせ

ご質問・ご不明点は GitHub リポジトリ の Issue よりお問い合わせください。
